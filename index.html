Skills

Projects

1. FileDumpling: A single page application that uses HTML, CSS, JavaScript, and Handlebars for the front-end, and Express.js, Node.js, Mongoose and MongoDB for the back-end. FileDumpling allows users to upload files of any extension and make it available for other users to download. The files are hosted by Amazon Web Services. FileDumpling was a team project that utilized an Agile SCRUM development framework, so we had daily stand ups and retros with our team to discuss our progress and review pull requests from overnight, and the plan/strategy to accomplish our goals for the day. We reviewed pull request and merged conflicts as a team.

2. Ate@: A full stack project built using Ruby on Rails for the API and JavaScript, HTML and CSS for the client side. A single page web application that allows users to save and retrieve their list of favorite restaurants. It has full user authentication and gives user full CRUD functions.

3. Tic-Tac-Toe: A single page application that utilizes a third party API to save and get game statistics. It has full user authentication, and allows a user to play a round of Tic-Tac-Toe with a friend.
